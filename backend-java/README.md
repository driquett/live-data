# Backend Java - Sistema de Gestión de Credenciales

Este módulo implementa la lógica del sistema usando Java y JDBC.

## Funcionalidades
- CRUD de usuarios
- Gestión de empleados
- Gestión de credenciales
- Conexión a MySQL

## Estructura
- config: conexión BD
- model: entidades
- dao: acceso a datos
- main: ejecución
